# celeborn → renamed to `celeborn-code`

**This project has moved.** The package formerly published here is now released as
[**celeborn-code**](https://pypi.org/project/celeborn-code/) — persistent memory for AI
coding agents, with the same `celeborn` CLI command.

```bash
pip install celeborn-code
# or
uv tool install celeborn-code
```

Installing this package (`celeborn`) still works — it is an empty redirect stub that
depends on `celeborn-code` — but please update your requirements to name
`celeborn-code` directly. No further releases will be published under the old name,
and all earlier `celeborn` releases (≤ 0.2.0) are yanked.
